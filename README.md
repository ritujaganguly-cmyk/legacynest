# LegacyNest

A lifetime care planning platform for families raising children with disabilities in India.

---

## What Is LegacyNest?

LegacyNest helps families build a complete, legally sound, and actionable care plan for a dependent child — so that when the primary caregiver is no longer able to provide care, nothing is left to chance.

The platform addresses a critical gap: over 80% of caregiving for persons with disabilities in India falls on immediate family, with no backup plan, no documentation, and no named successor.

---

## Core Features

| Module | Description |
|---|---|
| **Dashboard** | Unified care health score and progress tracker |
| **Child Profile** | Detailed disability profile, diagnosis, and care needs |
| **Parent Profile** | Primary and secondary caregiver records |
| **Care Circle** | Trusted network of backup caregivers and their roles |
| **Legal** | Guardianship, Will, Trust, and succession documents |
| **Financial** | Budget planning, insurance, government schemes, and asset tracking |
| **Medical** | Appointments, therapy sessions, and medication schedules |
| **Residential** | Housing options, transition planning, and placement records |
| **Emergency** | Emergency contacts, protocols, and crisis response plans |
| **Vault** | Secure document storage for all critical records |
| **Action Plan** | Prioritised to-do list generated from care plan gaps |
| **AI Assistant** | Contextual guidance powered by AI |
| **Notifications** | Live alerts for appointments, therapy, medications, and insurance |
| **Succession** | Named successor designation and handover documentation |

---

## Technology Stack

- **Frontend:** React 19, TypeScript, Vite 7
- **Routing:** TanStack Router
- **UI Components:** Radix UI, Tailwind CSS, shadcn/ui
- **Backend / Database:** Supabase (PostgreSQL + Auth + Storage)
- **State / Data:** TanStack Query, React Hook Form, Zod
- **PDF Generation:** jsPDF, html2canvas
- **Deployment:** Vercel
